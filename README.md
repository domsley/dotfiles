### Prerequisites:

- zsh
- [oh-my-zsh](https://ohmyz.sh/#install)
- Neovim (possibly nightly build)

### Setup

Run setup or link configs manually:

```shell
sh setup.sh
```

### Tmux

In tmux session hit 'prefix + I' to install tpm plugins.
